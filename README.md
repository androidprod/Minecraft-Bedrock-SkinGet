# Minecraft-Bedrock-SkinGet
MinecraftBedrockエディションでスキンを取得するサンプルです。

# 使用方法
1. node & npmをインストール
2. ```npm init -y```
  ※新しいフォルダーを作成することを推奨します 
3. ```npm install bedrock-protocol pngjs```
4. ```node index.js```
5. MinecraftでServerIPとPortを入力し、そのサーバーに接続してください。
6. Doneと出れば完了です。
